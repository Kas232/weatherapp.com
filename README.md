# 🌦 WeatherSphere Pro

WeatherSphere Pro is a modern weather forecasting web application built using **HTML, CSS, and JavaScript**. It provides real-time weather updates, 5-day forecasts, location-based weather information, and a responsive user interface powered by the OpenWeather API.

## 🚀 Features

* 🌍 Search weather by city name
* 📍 Get weather using current location
* 🌡 Real-time temperature updates
* 📅 5-Day weather forecast
* 💧 Humidity information
* 🌬 Wind speed details
* 📊 Pressure information
* 🌙 Dark Mode support
* ⭐ Favorite Cities shortcuts
* 🕒 Recent Search History
* 📱 Fully Responsive Design

## 🛠 Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)
* OpenWeather API
